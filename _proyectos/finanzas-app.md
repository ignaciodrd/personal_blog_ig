Este es el inicio de mi segundo proyecto: una app para manejar finanzas personales.

## Por Qué Este Proyecto

Las apps existentes son demasiado complicadas o cobran suscripciones caras. Quiero algo simple que se ajuste a mis necesidades.

## Funcionalidades Planeadas

- Registro de ingresos y gastos
- Categorías personalizables
- Gráficos de gastos mensuales
- Metas de ahorro

## Stack Tecnológico

Estoy considerando:
- React para el frontend
- Firebase para la base de datos
- Chart.js para visualizaciones
